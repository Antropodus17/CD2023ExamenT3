# Exámen CD T3

## Ejercicio 1 

Inserté un comentario de javadoc antes de cada método, atributo, clase y paquete para luego ejecutar los siguientes comandos dentro del proyecto.

```bash
 cd /rutadel proyecto

 mvn compile
 mvn javadoc:javadoc
```
La documentación se aloja en `//proyecto/target/site/apidocs`
